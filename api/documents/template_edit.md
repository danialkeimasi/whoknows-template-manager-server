# New Template

You can edit a template by using this route.

* #### URL

  /template/new

* #### Method:

  `POST`

* #### Data Params

     **Required:**

   `template`

* #### Success Response:

  * **Code:** 200 <br />
    **Content:** `[{...}, ...]`

* #### Error Response:

  * **Code:** 404 NOT FOUND <br />
    **Content:** `{ error : "User doesn't exist" }`
