# Question generate

Returns json data that contains a question generated by wanted request

* #### URL

  /question/generate

* #### Method:

  `POST`

* #### Data Params

     **Required:**

   `tags=[string]`
   `count=int`

* #### Success Response:

  * **Code:** 200 <br />
    **Content:** `[{...}, ...]`

* #### Error Response:

  * **Code:** 404 NOT FOUND <br />
    **Content:** `{ error : "User doesn't exist" }`
